# To Do List Web App

# The Project

This is a project created to run in the browser. The initial idea is a To Do List, or task list. The function of this project is to create lists where the user can add tasks and remove them. This project includes some goals such as:
1- real user validation using login and password,
2- the possibility to show only the lists created by the user with the option to edit and delete lists, as well as tasks,
3- the design of the site should provide a pleasant experience for users.
All of this is saved with the help of Firestore and Firebase.

## Technologies

* HTML
* CSS
* JavaScript
* Firebase
* Firestore

## How to use

The index.html file must be opened in the browser. After the page loads, the user will be able to log in with email and password. If the user does not have an account, they can create one. After logging in or signing up, the user is redirected to the page where lists can be created. Then, by selecting one of the created lists, the user can add tasks and delete them, and can also edit or delete the lists themselves.
