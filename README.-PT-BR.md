# To Do List Web App

# O Projeto

Este é um projeto criado para rodar no navegador, a ideia inicial é de um To Do List, ou, lista de tarefas. A função deste projeto é criar listas onde o usuário possa colocar tarefas e removê-las. Esse projeto envolve objetivos como 1- validação real de usuário por login e senha, 2- a possibilidade de mostrar apenas as listas que o usuário criou com a possibilidade de editar e excluir listas, assim como tarefas,3- o design do site deve trazer uma experiência agradável para os usuários.
Tudo isso é salvo com ajuda do Firestore e Firebase. 


## Tecnologias
- HTML
- CSS
- JavaScript
- Firebase
- Firestore

## Como usar
O index.html deve ser aberto no navegador, após o carregamento o usuário poderá fazer login com email e senha, caso não possua cadastro, é possível que ele se cadastre. Após o login ou cadastro, o usuário é redirecionado para a pagina em que as listas podem ser criadas, e em seguida, ao escolher uma das listas criadas, pode adicionar tarefas e exclui-las, assim como também pode editar e excluir as listas em si.
